# 🌍 Air Quality Index (AQI) Data Analysis using Python

> **Capstone Project** completed as part of the **AI Driven Data Analytics Course** by **Entri App**.

## 📌 Project Overview

Air pollution is one of the most critical environmental challenges affecting human health and ecosystems worldwide. Monitoring and analyzing air quality is essential for identifying pollution trends, understanding environmental conditions, and supporting informed decision-making.

This project focuses on analyzing an **Air Quality Index (AQI)** dataset using Python. It demonstrates the complete data analytics workflow—from data preprocessing and feature engineering to exploratory data analysis (EDA) and insightful visualizations.

---

## 🎯 Objectives

- Import and explore the AQI dataset
- Clean and preprocess raw data
- Handle missing and duplicate values
- Perform feature engineering
- Conduct Exploratory Data Analysis (EDA)
- Visualize pollution trends using various charts
- Analyze relationships between pollutants
- Generate meaningful insights from the data

---

## 📊 About the Dataset

The dataset contains air quality measurements collected from multiple monitoring stations.

### Features Included

| Feature | Description |
|----------|-------------|
| AQI | Air Quality Index |
| PM2.5 | Fine Particulate Matter (≤2.5 μm) |
| PM10 | Particulate Matter (≤10 μm) |
| CO | Carbon Monoxide |
| SO₂ | Sulfur Dioxide |
| NO₂ | Nitrogen Dioxide |
| O₃ | Ground-Level Ozone |
| Date | Observation Date |
| Time | Observation Time |
| Monitoring Station | Air Quality Monitoring Station |
| AQI Category | Air Quality Classification |

# 🔍 Key Insights

- Air quality varies significantly across monitoring stations.
- PM2.5 has a strong impact on overall AQI.
- Pollution levels change across different months and seasons.
- Moderate and Poor AQI categories occur more frequently than Good air quality.
- Correlation analysis reveals strong relationships among several pollutants.
- Visualizations help identify pollution hotspots and seasonal trends.
